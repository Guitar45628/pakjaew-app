# 🎙️ ปากแจ๋ว (PakJaew App) — ระบบแสดงผลการโดเนทพร้อมเสียงสังเคราะห์บอกอารมณ์สุดเจ๋ง!

ยินดีต้อนรับสู่ **ปากแจ๋ว (PakJaew App)** แพลตฟอร์มแจ้งเตือนยอดโดเนท (Donation Alert Overlay) สำหรับสตรีมเมอร์ผ่านโปรแกรม OBS Studio ที่ออกแบบมาด้วยดีไซน์ Obsidian Midnight สุดมินิมอล พร้อมพลังสมองของ **Gemini 3.1 Flash Text-to-Speech** ที่สามารถแปลงข้อความโดเนทเป็นเสียงพูดภาษาไทยที่วิเคราะห์บอกอารมณ์ตามข้อความได้อย่างสมจริงที่สุด! (โกรธจัด, เศร้าสะอื้น, ตื่นเต้นตลก หรือหล่อเท่คูลๆ)

---

## ✨ คุณสมบัติเด่น (Features)
* 🖤 **Obsidian Midnight Design:** ดีไซน์การ์ดแสดงผลยอดโดเนทสีดำขอบมนแบบกระจกเงา ดูพรีเมียม สไตล์ Shadcn UI ยอดนิยม อ่านง่ายสบายตาบนจอ OBS
* 🤖 **Gemini 3.1 Flash TTS Engine:** ใช้ AI สังเคราะห์เสียงโมเดลล่าสุดของ Google มอบความรู้สึกและสำเนียงการออกเสียงภาษาไทยสมจริงที่สุด เปลี่ยนสำเนียงอารมณ์และระดับเสียงได้ทันทีตามประเภทของการโดเนท
* 🎙️ **Microsoft Edge-TTS (ฟรี 100%):** เสียงพูดสังเคราะห์ภาษาไทยดั้งเดิมจากฝั่ง Microsoft ฟรีทั้งหมด 100% โดยไม่ต้องใช้บัตรเครดิต หรือกรอก API Key ใดๆ
* 💸 **100% Free & Low-Cost (ประหยัดค่าใช้จ่าย):** รองรับ Gemini 3.1 Flash ในแผนฟรี (Free Tier) ช่วยให้คุณได้ระบบสังเคราะห์เสียงบอกอารมณ์สุดล้ำโดยไม่ต้องเสียเงินสักบาท
* ⚡ **Lightweight & Real-time (เน้นเล็ก เบา และทำงานเรียลไทม์):** รับส่งข้อมูลและจัดเรียงคิวเสียงแบบเรียลไทม์ผ่าน In-Memory Queue เซ็ตอัพง่ายและเปิดรันระบบเล่นได้ทันที โดยไม่ต้องตั้งค่าหรือเตรียมระบบฐานข้อมูลให้ยุ่งยากวุ่นวายในขั้นตอนเริ่มต้น

## 🚀 วิธีการติดตั้งและเริ่มเล่นทันที (Quick Start)

สามารถเลือกวิธีติดตั้งและเปิดใช้งานได้ง่ายๆ 2 ช่องทาง ดังนี้:

### ช่องทางที่ 1: รันผ่าน Docker (แนะนำสำหรับทุกคน - ง่ายที่สุดตัวเดียวจบ) 🐳
คุณต้องติดตั้ง Docker และ Docker Compose ในเครื่องก่อน จากนั้นทำตามขั้นตอนด้านล่างนี้:

1. **เปิดโฟลเดอร์หลังบ้านสร้างคีย์ API:**
   เปิดไฟล์ `backend/.env` (หากไม่มีให้สร้างขึ้นมา) แล้วใส่คีย์ Gemini API ของคุณดังนี้:
   ```env
   GEMINI_API_KEY=ใส่คีย์_Gemini_API_จริงของคุณตรงนี้
   ```
   > [!TIP]
   > สามารถขอคีย์ Gemini API ได้ฟรีผ่านหน้าเว็บ [Google AI Studio](https://aistudio.google.com) ครับ!

2. **สั่งรันระบบด้วยคำสั่งเดียว:**
   เปิด Terminal ที่ตำแหน่งโฟลเดอร์หลักของโปรเจกต์นี้ แล้วรันคำสั่ง:
   ```bash
   docker-compose up --build
   ```

3. **เข้าใช้งานระบบ:**
   * **หน้าจอแอดมินผู้จัดการระบบ:** เข้าไปที่ [http://localhost:5173/admin](http://localhost:5173/admin)
     * *ชื่อเข้าใช้ (Username):* `admin`
     * *รหัสผ่าน (Password):* `admin1234`
   * **หน้าจอแสดงผล OBS Overlay:** นำลิงก์นี้ไปใส่ใน Browser Source ของโปรแกรม OBS Studio:
     `http://localhost:5173/overlay?key=pakjaew_super_secret_token_2026`

---

### ช่องทางที่ 2: รันแบบ Manual (สำหรับนักพัฒนาและผู้ใช้งานทั่วไปในเครื่องตัวเอง) 💻

#### 1. ส่วนของหลังบ้าน (Backend - FastAPI):
1. ตรวจสอบให้แน่ใจว่าติดตั้ง [Python 3.10+](https://python.org) แล้ว
2. เปิด Terminal ในตำแหน่งโฟลเดอร์ `backend/` แล้วติดตั้ง Library ทั้งหมด:
   ```bash
   cd backend
   pip install -r requirements.txt
   ```
3. คัดลอกและสร้างไฟล์ `.env` สำหรับเก็บคีย์ API:
   * ทำการสร้างไฟล์ชื่อ `.env` ไว้ในโฟลเดอร์ `backend/`
   * ใส่ข้อความ: `GEMINI_API_KEY=คีย์_Gemini_API_จริงของคุณ`
4. สั่งรันเซิร์ฟเวอร์หลังบ้าน:
   ```bash
   uvicorn app.main:app --reload --port 8000
   ```

#### 2. ส่วนของหน้าบ้าน (Frontend - React Vite):
1. ตรวจสอบให้แน่ใจว่าติดตั้ง [Node.js](https://nodejs.org) หรือ [Bun](https://bun.sh) แล้ว
2. เปิด Terminal ในตำแหน่งโฟลเดอร์ `frontend/` แล้วติดตั้ง Dependency และสั่งรัน:
   * **กรณีใช้ Bun (แนะนำ):**
     ```bash
     cd frontend
     bun install
     bun dev
     ```
   * **กรณีใช้ npm:**
     ```bash
     cd frontend
     npm install
     npm run dev
     ```
3. บราวเซอร์จะเปิดหน้าเว็บที่พอร์ต [http://localhost:5173](http://localhost:5173) อัตโนมัติ!

## 🎲 วิธีการทดลองและเล่นฟีเจอร์เด่น!

1. เข้าหน้าจอแอดมินที่ [http://localhost:5173/admin](http://localhost:5173/admin) (เข้าสู่ระบบด้วย `admin` / `admin1234`)
2. เปิดหน้าจอเบราว์เซอร์สำหรับจำลองผล OBS เพิ่มขึ้นมาอีกหนึ่งหน้าต่างที่:
   👉 [http://localhost:5173/overlay?key=pakjaew_super_secret_token_2026&demo=true](http://localhost:5173/overlay?key=pakjaew_super_secret_token_2026&demo=true)
   > [!NOTE]
   > การเติมตัวเลือก `&demo=true` จะแสดง **ปุ่มลอง Emotion จำลองด้านล่าง** และใส่พื้นหลังมืด เพื่อให้พี่ทดสอบการแสดงผลและเสียงได้ง่ายขึ้นโดยตรงผ่านเว็บเบราว์เซอร์ของพี่เองโดยไม่ต้องเปิดโปรแกรม OBS!
3. สลับการตั้งค่าระบบเสียง (TTS Engine) เป็น **Gemini-TTS 🤖** บนหน้าแผงควบคุมแอดมิน
4. กดปุ่ม **ทอยลูกเต๋า 🎲** บนหน้าแอดมิน เพื่อสุ่มยิงยอดโดเนทแปลกๆ ขึ้นสู่หน้าจอและรอฟังเสียงพูดภาษาไทยสังเคราะห์ตามอารมณ์สุดล้ำได้เลยครับ!

## 📂 โครงสร้างโฟลเดอร์และไฟล์สำคัญ (Project Structure)
```
pakjaew-app/
├── backend/                  # เซิร์ฟเวอร์ API หลังบ้าน (FastAPI)
│   ├── app/
│   │   ├── api/              # เส้นเชื่อมต่อ API และ WebSocket
│   │   ├── workers/          # Queue Worker สำหรับจัดเรียงและเจนเสียง TTS
│   │   └── main.py           # จุดเริ่มต้นในการเปิดเซิร์ฟเวอร์หลังบ้าน
│   ├── Dockerfile
│   └── requirements.txt      # รายการ Library ของฝั่ง Python (FastAPI, google-genai, etc.)
├── frontend/                 # ระบบแสดงผลและแผงควบคุมหน้าบ้าน (React Vite)
│   ├── src/
│   │   ├── pages/
│   │   │   ├── AdminPage.tsx # แผงควบคุมสลัด Shadcn และเครื่องสุ่มโดเนทลูกเต๋า
│   │   │   └── OBSOverlayPage.tsx # หน้าจอแสดงผลดีไซน์ Obsidian Midnight ดำหรู
│   └── Dockerfile
├── docker-compose.yml        # แฟ้มเชื่อมบริการ Docker รวมเป็นหนึ่งคำสั่ง
└── README.md                 # คู่มือแนะนำการเล่นและติดตั้งฉบับเข้าใจง่ายที่สุด
```
